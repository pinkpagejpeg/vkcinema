# Запуск проекта

1. Клонируйте репозиторий:
  ```bash
  git clone https://github.com/pinkpagejpeg/vkcinema.git
  cd your-repository-name
   ```

2. Установите зависимости:
  ```bash
  npm install
  ```

3. Добавьте .env файл в корень проекта:
  ```bash
  VITE_TOKEN=YOUR_API_TOKEN
  ```

4. Запустите проект
  ```bash
  npm run dev
  ```

Откройте в браузере:
Клиент: http://localhost:5173
